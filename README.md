Project Overview:
An interactive Power BI dashboard designed to analyze and uncover insights from Netflix’s global content library. This project transforms raw entertainment data into a comprehensive visual story, tracking performance across genres, ratings, countries, directors, and release years to help understand audience preferences and production strategies.

Key Features and Insights:
1. Content Overview: Analyzes 400 titles with an average IMDb rating of 7.07, providing a snapshot of Netflix’s catalog quality.
2. Genre Distribution: Breaks down titles across Action, Thriller, Comedy, Drama, Horror, Romance, Documentary, and Sci‑Fi, highlighting the diversity of Netflix’s storytelling.
3. Ratings Analysis: Evaluates maturity segmentation with content spread across G, PG, PG‑13, TV‑14, TV‑MA, and R, offering insights into audience demographics.
4. Director Contributions: Showcases leading directors such as Carlos Diaz, Anita Kapoor, and David Brown, each contributing significantly to genre outputs.
5. Country Insights: Maps content contributions from major regions including the UK, France, Spain, US, Canada, India, and South Korea, revealing Netflix’s global production footprint.
6. Release Year Trends: Tracks content growth from 2000 to 2025, identifying peaks in production and evolving audience demand.
7. Type Analysis: Differentiates between Movies and TV Shows, enabling deeper segmentation of Netflix’s catalog.

Project Objective:
The goal of this dashboard is to provide stakeholders with actionable intelligence about Netflix’s content strategy. By identifying genre popularity, understanding regional contributions, and analyzing rating distributions, this tool helps optimize content planning, audience targeting, and global expansion strategies.

Tools and Skills Demonstrated:
1. Data Preparation & Cleaning: Managed in Microsoft Excel to preprocess and structure the dataset.
2. Data Visualization & Storytelling: Built in Power BI with interactive slicers, custom branding, and intuitive chart designs to maximize insights.

How to View the Project:
1. Download the .pbix file from this repository.
2. Open it using Power BI Desktop.
3. Interact with slicers (Genre, Country, Rating, Director, Release Year) to dynamically explore the data.
